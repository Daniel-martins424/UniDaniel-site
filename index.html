O que é linguagem de marcação?

Uma linguagem de marcação é um sistema para adicionar informações estruturais e de apresentação a um documento de texto, geralmente através de tags ou códigos que indicam a função de diferentes partes do texto. Ela não é uma linguagem de programação no sentido de realizar cálculos ou ações, mas sim de descrever e organizar o conteúdo.

Exemplos:

  • Definir a estrutura do documento: Separar títulos, parágrafos, listas, etc.
  • Especificar a apresentação: Definir formatação como itálico, negrito, cores, etc.
  • Fornecer metadados: Informações adicionais sobre o documento, como autor, data de criação, etc.
  • Facilitar o processamento: Permite que programas processem e exibam o conteúdo de forma consistente e eficiente.
